## 🎓 Learning Path

### Beginner (Just starting)
```
1. README.md (Overview)
2. docs/SETUP.md (Installation)
3. docs/FAQ.md (Q&A)
```
**Time**: 15 minutes
**Result**: Blueprints imported and first automation created

---

### Intermediate (Want to understand blueprints)
```
1. docs/BLUEPRINT_GUIDE.md (How blueprints work)
2. docs/TROUBLESHOOTING.md (When things go wrong)
3. Review example blueprints
```
**Time**: 30-45 minutes
**Result**: Understand blueprint structure and troubleshoot issues

---

### Advanced (Creating your own)
```
1. docs/BLUEPRINT_GUIDE.md (Complete reference)
2. CONTRIBUTING.md (Submission guidelines)
3. blueprints/automation/motion_light.yaml (Study example)
4. Create and test your blueprint
5. Submit PR with contribution
```
**Time**: 2-4 hours per blueprint
**Result**: Published blueprints in repository
